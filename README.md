# Summerlog

An AI-powered log summarizer for Docker containers.

## Installation

To install the development version directly from GitHub:

```bash
pip install git+https://github.com/allisonhere/summerlog.git
```

## Configuration

After installation, run the interactive configuration tool:

```bash
summerlog-configure
```

## Usage

To generate a log summary:

```bash
summerlog
```
